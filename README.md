# weather-coding-test
Simple python coding test


## Environment
This exercise requires using `pipenv` as the Python dependency manager. On iOS, you can install that with command:
```
brew install pipenv
```

## Dependencies
To install all code dependencies, you will need to use `pipenv` with command:
```
pipenv install -d
```

## Unit tests
After all dependencies are installed, you can run unit tests with command:
```
pipenv run pytest -v
```

## Application
To run the application code use command:
```
pipenv run python src/coding_test.py
```
